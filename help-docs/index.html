Overview Of Template

MERA Template is best template for Showcasing your skill to your visitors, it Comes with four Color options that you can choose from pink , brown, blue and green. You can use any color which you like. You just have to put the respective style sheet link in header and you are set to go.By default we are using pink.css as the base style you can replace it if you wish.
